About This Tool

This Spam Detector uses a Machine Learning model (Naive Bayes with TF-IDF vectorization) to classify messages as "Spam" or "Not Spam".

Features:

  • Single Message Check: Get an instant prediction for any message.
  • Batch Prediction: Check multiple messages at once by pasting text or uploading a file.
  • Confidence Score: See the probability of a message being spam.
  • User Feedback: Help improve the model by marking predictions as correct or incorrect (feedback helps us refine future versions!).
  • API Available: A /api/predict endpoint is available for programmatic access (POST request with JSON {"message": "your message here"}).

The model was trained on a dataset of SMS messages and continuously learns from feedback.

Current model version: 1.1.0 (TF-IDF, Feedback Enabled)
